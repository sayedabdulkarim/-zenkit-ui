# ZenKit UI

Welcome to **ZenKit UI** - A modern React component library built on ZenKit CSS.

## Features

- **Zero-runtime CSS** - Built on ZenKit CSS for optimal performance
- **Fully typed** - Written in TypeScript with complete type definitions
- **Accessible** - WCAG 2.1 AA compliant components
- **Customizable** - CSS Variables for easy theming
- **Tree-shakeable** - Import only what you need
- **100+ Components** - Comprehensive component library

## Installation

```bash
npm install @zenkit-ui/core zenkit-css
# or
pnpm add @zenkit-ui/core zenkit-css
```

## Quick Start

```tsx
// Import ZenKit CSS in your app's entry point
import 'zenkit-css/dist/zenkit.css';

// Use components
import { Button } from '@zenkit-ui/core';

function App() {
  return (
    <Button variant="solid" colorScheme="primary">
      Click me
    </Button>
  );
}
```

## Packages

### @zenkit-ui/core

Core components including:

**Layout & Structure**
- AppShell, Container, Grid, Flex, Stack, Group, Box, Center, AspectRatio

**Navigation**
- Navbar, Sidebar, Breadcrumb, Menu, Tabs, Pagination, Steps, NavLink

**Data Entry**
- Input, Select, Checkbox, Radio, Switch, Slider, DatePicker, Upload, Form

**Data Display**
- Table, DataGrid, Card, Avatar, Badge, Tag, Timeline, Tree, Descriptions

**Feedback**
- Alert, Modal, Drawer, Toast, Notification, Progress, Skeleton, Spinner

**Overlay**
- Popover, Tooltip, Dropdown, Dialog, Sheet, Popconfirm

**Utility**
- Portal, Transition, ConfigProvider, FloatButton, Affix, VirtualList

### @zenkit-ui/hooks

Reusable React hooks for building accessible UI components:

- `useDisclosure` - Modal/dropdown open/close state
- `useClickOutside` - Click outside detection
- `useFocusTrap` - Keyboard focus trapping
- `useControllable` - Controlled/uncontrolled component state
- `useMediaQuery` - Responsive breakpoint detection
- `useKeyboard` - Keyboard event handling
- `useClipboard` - Copy to clipboard functionality
- `useLocalStorage` - Persistent local storage state
- `useDebounce` - Debounced value updates
- `useIntersection` - Intersection observer hook
